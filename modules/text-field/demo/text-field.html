Text fields

lx-text-field is an input directive that will become an input element or a textarea element according to the transcluded element.

Parameter Type Description
lx-error boolean When present, set red error style.
lx-fixed-label boolean When present, labels will act like classic placeholders.
lx-icon string Add an icon next to the text input. Input must have a fixed label in this case.
lx-label string The label of the input displayed as a placeholder (required).
lx-theme string Set this option to dark if inputs are displayed into a dark box. Options: light (default), dark
lx-valid boolean When present, set green valid style.
