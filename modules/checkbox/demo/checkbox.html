Checkbox

lx-checkbox is a checkbox directive that will become a checkbox input element.

You can specify the checkbox label as transcluded element.
If you want to specify a helper message, you will need to transclude two directives.
lx-checkbox-label for the label and lx-checkbox-help for the helper message. Disabled style will automaticly be applied when using ng-disabled or disabled attribute. You can specify checkbox color with main colors defined in dist/scss/core/settings/defaults.scss
Parameter Type Description
lx-color string Define the button color. Options: colors defined in colors and sizes section in dist/scss/core/settings/defaults.scss
