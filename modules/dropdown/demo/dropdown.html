Dropdown

lx-dropdown is a directive that create a dropdown menu.

You need to transclude two directives : lx-dropdown-toggle that transclude the toggle button and lx-dropdown-menu that transclude the menu content.
Parameter Type Description
lx-escape-close boolean Specify if the dropdown should be closed with escape key. Default: true
lx-over-toggle boolean Specify if the dropdown should be opened over the toggle button.
lx-position string Define the position of the dropdown menu. Options: left (default), right
lx-width string Define the dropdown menu width in pixels. If a percent value is defined, the dropdown menu width will be relative to the width of the toggle button.
